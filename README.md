# TypeCast 🚀

An elegant, production-ready AI-powered Resume Builder designed to transform raw career details into stunning, ATS-friendly resumes in seconds. Built with **Next.js 14**, **TypeScript**, and **Tailwind CSS**, it leverages advanced AI integrations to provide tailored content suggestions, layout optimization, and real-time formatting.

---

## ✨ Features

- **AI-Powered Content Generation:** Instantly polish job descriptions, rewrite bullet points for high impact, and generate professional summaries tailored to specific target roles.
- **Dynamic Live Preview:** Real-time side-by-side rendering of your resume changes using structured TSX templates.
- **ATS Optimization:** Intelligent feedback on resume keyword density and formatting to ensure compatibility with Applicant Tracking Systems.
- **Modern Responsive UI:** A gorgeous, minimalist dark-mode-first user interface built with Tailwind CSS and Shadcn UI components.
- **Export to PDF:** Clean, single-page or multi-page high-fidelity PDF exports with preserved printing layout margins.

---

## 🛠️ Tech Stack

- **Framework:** [Next.js 14](https://nextjs.org/) (App Router)
- **Language:** [TypeScript](https://www.typescriptlang.org/) (Strictly Typed)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/) & [Shadcn UI](https://ui.shadcn.com/)
- **State Management:** Redux Toolkit / React Context
- **AI Integration:** OpenAI API / Anthropic SDK
- **Database / Cache:** Prisma ORM & Redis (for prompt rate-limiting)
